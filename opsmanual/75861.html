A5.15 Applicants not considered to be of good character for a residence class visa (19/11/2010)

Applicants not considered to be of good character for a residence class visa are classified as follows:

  1. applicants who will not be granted a residence class visa (see A5.20); or
  2. applicants who will not normally be granted a residence class visa (see A5.25) unless a character waiver is granted; or
  3. applicants whose applications for a residence class visa will usually be deferred (see A5.35).

Effective 29/11/2010
