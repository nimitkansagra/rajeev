E4.15 Where to lodge an application

  1. Applications should be lodged at the NZIS branch, MFAT post or certain specifed British posts responsible for receiving applications of that type from the geographical area or country in which the applicant currently lives. A list of current Receiving Offices is contained in Appendix 4.
  2. An application can be lodged at any NZIS branch or MFAT post on the current list of Receiving Offices. However, if the application is not lodged at the appropriate office, the application may be referred to the appropriate office.

Effective 26/07/1999
